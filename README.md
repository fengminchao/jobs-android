# Jobs

## 介绍

Jobs 是一款汇集了湖北各大高校的招聘宣讲信息，帮助用户管理招聘信息的应用，实现了头像上传，夜间主题无缝切换等功能。应用 MVP+Retrofit+Dagger2+RxJava实现。
